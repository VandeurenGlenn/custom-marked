> A command line interface for fast es6 development

## Installation

```sh
$ npm install --save --global custom-marked
```

## requirements

### [backed-cli](https://github.com/vandeurenglenn/backed-cli)
```sh
  npm install --global backed-cli
```

### Commands

#### Build[--build, -b]
```sh
  backed --build
```

#### Serve[--serve, -s]
```sh
  backed --serve
```
